# Text-to-Speech Test

Simple TTS scripts using Gemini and Groq APIs. Convert text to natural-sounding speech with different voices. Easy setup with environment variables.

## Usage
- `TTS.py` - Gemini TTS with Zephyr voice
- `playtts.py` - Groq TTS with Fritz voice
